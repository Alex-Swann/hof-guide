About HOF

What is HOF?

HOF (Home Office Forms) is a framework designed to assist developers in creating form-based workflows in a rapid, repeatable and secure way. It aims to reduce simple applications as much as possible to being configuration-only.

About this documentation

This documentation is intented for developers and technical architects who want to build applications using the HOF framework.

If you have questions which are not covered by this documentation then there is a #hof channel in the UK Government Digital slack channel which can be accessed by anyone with a .gov.uk email address.
